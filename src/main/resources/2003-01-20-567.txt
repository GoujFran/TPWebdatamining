 Non seulement les météorologistes bénéficient de calculateurs surpuissants, mais en prime ils compilent des archives fort bien tenues. Ce qui permet de dénicher quelques données significatives relevées par les stations fixes situées entre le niveau de la mer et 1.600 m d'altitude.  Température maxi absolue : Le Luc (Var) 42,7 ; Luxeuil 38,9 le 31 juillet 1987 ; Besançon 38,8 ; Colmar 38 ; Nancy 37,6 ; Belfort 36,3.  Température mini absolue : Luxeuil - 25,2 le 5 janvier 1985 ; Colmar - 24,8 ; Nancy - 21,6 ; Dijon - 21,3 ; Besançon - 20,7 ; Belfort - 20,6.  Vent vitesse absolue (en mètres par seconde) : Pointe du Raz 72 (soit 259 km/h) ; Mont Aigoual 61 ; Langres 45 ; Belfort 43 ; Metz 40 ; Colmar 38 ; Nancy 38 ; Luxeuil 35 (ouragan de décembre 1999) ; Dijon 31.  Température moyenne sur une année : Corse 16,3° ; Toulon 15,9 ; Lyon 11,4 ; Dijon 10,5 ; Colmar 10,2 ; Luxeuil 9,5 ; Langres 8,9.  Insolation moyenne (heures de soleil en une année) : Toulon 2.889 ; Lyon 1.975 ; Besançon 1.871 ; Dijon 1.831 ; Langres 1.754 ; Luxeuil 1.732.  Pluviométrie moyenne (mm/m2) : Mont Aigoual 2.194 ; Belfahy 1.968 ; Biarritz 1.636 ; Dax 1.354 ; Besançon 1.295 ; Belfort 1.257 ; Luxeuil 1.171 ; Frotey-lès-Vesoul 1.007 ; Dijon 866.
