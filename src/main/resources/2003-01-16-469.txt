 On a toujours attribué aux fleurs des significations particulières. En voici les plus connues.  Achillée : guérison, guerre, mérite caché.  Amarante : indifférence.  Anémone : abandon, amour trahi.  Aster : doute, méfiance.  Bleuet : délicatesse.  Bouquet panaché : galanterie.  Campanule : propositions galantes mais futiles.  Chardon : austérité.  Chrysanthème : amour sous tous ses aspects.  Dahlia : reconnaissance, présage, faux espoirs.  Gueule de loup : caprice.  Guirlande : chaîne d'amour.  Hortensia : froideur, séparation.  Jacinthe : bienveillance, grâce, douceur.  Jonquille : désir.  Lis : majesté, pureté.  Mimosa : souvenir.  Narcisse : amour, égoïsme, considération.  Nigelle : lien d'amour.  Oeillet : affection sincère.  Oeillet blanc : fidélité, loyauté.  Oeillet jaune : dédain.  Oeillet rose : amour ardent et sincère.  Orchidée : habileté.  Pavot : lenteur, doute, oubli, surprise.  Pavot rouge : orgueil, beauté éphémère.  Pivoine : timidité, confusion.  Rose blanche : silence.  Rose jaune : infidélité, honte.  Rose orangée : tendresse.  Rose rose : franchise, tendresse.  Rose rouge : passion.  Tournesol : fausse richesse.  Tulipe : honnêteté, message d'amour, magnificence.  Véronique : fidélité.
