 Le club de judo, sous la houlette des entraîneurs Pascal, Dominique et Michel Mouillet, a réussi une très belle performance lors du 13e tournoi pour le challenge Maurice-Eltgen à Audincourt.  Lors des deux journées de compétition, le club s'est classé 5e sur 49 équipes engagées. Ils ont reçu une dédicace et les félicitations de Stéphane Traineau, champion du monde et d'Europe, aujourd'hui directeur des équipes de France.  A signaler que les mini-poussins, poussins et benjamins ont terminé 2e sur 49.  - Les résultats, mini-poussins, mini-poussines : Sébastien Crolotte 2 ; Jocelyn Figini 4e ; Mathieu Genin 4e ; Roman Levin 2e ; Emeline Oudet 4e ; Kevin Parent 2e ; Romain Ressot 2e ; Julian Richer 3e ; Cyril Vadam 3e.  - Poussins, poussines : Lena Clavaud 4e ; Julia Duplessis 3e ; Tanguy Garmi 1er ; Loïc Girardot 4e ; Marion Juillerat 3e ; Matthieu Molle 2e ; Alexandre Oudet 1er ; Joffrey Ruez 1er.  - Benjamins, benjamines : Mathieu Bisoffi 2e ; Jonathan Buffa 4e ; Juliette Clavaud 3e ; Joany Cmas 3e ; Guillaume Crolotte 4e ; Marina Pastorino 2e ; Aurélien Tissot 3e.  - Minimes : Anthony Buffa 4e.  - Cadets : Romain Ruspini 3e.  Les entraînements se poursuivent pour ces jeunes dans un grand sérieux et avec le soutien de leurs parents.
